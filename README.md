# catculator

고양이는 생애주기, 몸무게, 현재 상태(중성화 여부, 임신 여부 등) 등에 따라 적정량의 사료를 섭취해야 합니다. </br>
**catculaotr** 서비스는 집사를 돕기 위해 만들어진 고양이 사료량 계산 서비스입니다.
