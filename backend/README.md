## Installation

```bash
# build application
$ yarn build

# execute application on local environment
$ yarn dev

# execute application
$ yarn start

# execute test code
$ yarn test

# execute test code coverage
$ yarn test:cov

# execute test code summary coverage
$ yarn test:sum

# make test code badge
$ yarn test:bad

# linting
$ yarn lint

# make docker images
$ make dbuild

# execute docker service
$ make drun

# remove docker service
$ make drm
```
